Con este plugin podrás adecuar tu web corporativa o blog a la LSSI y LOPD y crear tus textos legales. Plugin cookies Wordpress.

== Description ==

Cumplirás con la normativa sobre cookies y te creará automaticamente tus textos legales de privacidad, cookies y el Aviso Legal con tu identificación y condiciones de uso de tu web. Podrás personalizar los textos libremente editando de forma simple la página.

Nuestros abogados especializados en nuevas tecnologías recogen las exigencias legales en una solución de fácil uso e instalación.

La forma más automática de adaptar tu web a los requisitos legales.

Este plugin no está elaborado para adecuar legalmente tiendas online que realicen e-commerce.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/legalplus` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Legal+ Name screen to configure the plugin

== Frequently Asked Questions ==

= Se crean automáticamente las páginas legales? =

Si, al activar el plugin se crear las paginas de política de privacidad y cookies, y Aviso legal

= Donde están las páginas legales? =

En la lista general de páginas de wordpress

= Puedo personalizar el contenido? =

En caso de tener alguna sección que no necesites, o simplemente quieras cambiar, las páginas se pueden personalizar como cualquier otra página de wordpress

= Que tengo que hacer despues de instalar y activar el plugin? =

Rellenar los datos generales sobre tu web, que se utilizarán automáticamente en las páginas
